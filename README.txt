Description
The Mobile Selling Application is a simple web-based system that allows users to browse and purchase mobile devices. It also includes a "Contact Us" feature for users to send inquiries or feedback.

Features
User authentication (login and logout).
Browse and purchase mobile devices.
Contact Us form for user inquiries and feedback.
Basic user interface using HTML and Java servlets.
Integration with a MySQL database for user data storage.
Prerequisites
Java Development Kit (JDK)
MySQL database
Web server (e.g., Apache Tomcat) for Java servlets
HTML/CSS for front-end presentation
Installation
Clone the project repository:

bash
Copy code
git clone https://github.com/your-username/mobile-selling-app.git
Create the MySQL database and import the schema from the provided SQL file.

Configure the database connection by updating the DB_URL, DB_USER, and DB_PASSWORD values in your Java code.

Deploy the Java servlets to your web server.

Access the application by navigating to the appropriate URL in your web browser.

Usage
Home Page: Browse and purchase mobile devices.

Contact Us Page: Use the "Contact Us" page to send inquiries and feedback.

Login and Logout: Authenticate using the login page and log out when done.
